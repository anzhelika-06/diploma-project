## 📖 Описание проекта

EcoSteps - это веб-приложение для расчета и мониторинга углеродного следа пользователей. Платформа позволяет:
- Рассчитывать углеродный след на основе данных о питании, транспорте, энергопотреблении
- Получать персонализированные рекомендации по снижению выбросов CO₂
- Отслеживать прогресс и получать достижения
- Создавать команды и делиться историями успеха
- Просматривать статистику и аналитику

## 🚀 Установка и запуск

### Шаг 1: Установка Docker Desktop

1. Скачайте Docker Desktop с официального сайта: https://www.docker.com/products/docker-desktop/
2. Запустите установщик и следуйте инструкциям
3. После установки перезагрузите компьютер
4. Запустите Docker Desktop и дождитесь его полной загрузки (иконка Docker в трее должна быть зеленой)

### Шаг 2: Получение проекта

**Вариант А: Если у вас есть архив проекта**
1. Распакуйте архив в любую папку (рекомендуется не на диск C:)
2. Откройте папку проекта в Visual Studio Code

**Вариант Б: Клонирование через Git**
```bash
git clone https://github.com/anzhelika-06/diploma-project.git
```
Затем откройте папку `diploma-project` в Visual Studio Code

### Шаг 3: Запуск проекта через терминал VS Code

1. В Visual Studio Code откройте встроенный терминал:
   - Нажмите `Ctrl + ~` (тильда) или
   - Меню: `Вид` → `Терминал` или
   - `Terminal` → `New Terminal`

2. Убедитесь, что вы находитесь в папке `diploma-project`:
```bash
cd diploma-project
```

3. Выполните команду для запуска всех сервисов:
```bash
docker-compose up --build -d
```

### Шаг 4: Автоматическая установка зависимостей

После выполнения команды Docker автоматически:
1. **Загрузит базовые образы** (Node.js, PostgreSQL, Redis) - при первом запуске ~5-10 минут
2. **Создаст контейнеры** для каждого сервиса (клиент, сервер, база данных, кэш)
3. **Установит все библиотеки** внутри контейнеров:
   - Frontend: React, Vite, Socket.IO Client, Recharts и другие (~150 MB)
   - Backend: Express, PostgreSQL, Redis, Socket.IO, bcrypt и другие (~200 MB)
4. **Инициализирует базу данных** с тестовыми данными (35 пользователей, команды, достижения)

**Важно**: Все зависимости устанавливаются автоматически внутри Docker контейнеров. Вам не нужно устанавливать Node.js, npm или какие-либо библиотеки вручную на ваш компьютер!

Дождитесь завершения процесса (в терминале появится сообщение о успешном запуске).

### Шаг 5: Проверка работы

1. Откройте браузер
2. Перейдите по адресу: **http://localhost:5173**
3. Вы должны увидеть главную страницу EcoSteps

**Доступные сервисы:**
- 🌐 **Клиент (React)**: http://localhost:5173
- 🔧 **API Server**: http://localhost:3001
- 🗄️ **PostgreSQL**: localhost:5432
- 💾 **Redis**: localhost:6379

## 🛠 Используемые технологии и библиотеки

### Backend (Серверная часть)
- **Node.js + Express** - серверная платформа и веб-фреймворк
- **PostgreSQL** - реляционная база данных для хранения пользователей, постов, команд
- **Redis** - кэширование данных и управление WebSocket сессиями
- **Socket.IO** - real-time обновления (уведомления, онлайн статусы)
- **bcrypt** - безопасное хеширование паролей
- **JWT (jsonwebtoken)** - токены для аутентификации пользователей
- **node-cron** - планировщик задач (ежедневные эко-советы)
- **express-rate-limit** - защита от DDoS атак

### Frontend (Клиентская часть)
- **React** - библиотека для построения интерфейса
- **Vite** - быстрая сборка и разработка
- **React Router** - навигация между страницами
- **Socket.IO Client** - получение обновлений в реальном времени
- **Recharts** - графики и диаграммы для статистики
- **Material Icons** - иконки интерфейса

### Инфраструктура
- **Docker + Docker Compose** - контейнеризация и оркестрация всех сервисов
- **PostgreSQL 15** - база данных (27 таблиц, 7 представлений, 11 функций/процедур)
- **Redis 7** - кэш и хранилище сессий
- **Nginx** - reverse proxy для продакшена

**Все библиотеки устанавливаются автоматически** при запуске Docker контейнеров. Общий размер зависимостей: ~350 MB.

## 📝 Разработка

### Запуск без Docker (для разработки)

Если вы хотите запустить проект локально без Docker:

1. Установите Node.js 18+ с https://nodejs.org/
2. Установите PostgreSQL 15 и Redis 7
3. Установите зависимости:

```bash
# Установка зависимостей клиента
cd diploma-project/client
npm install

# Установка зависимостей сервера
cd ../server
npm install
```

4. Настройте переменные окружения в `server/.env`
5. Запустите сервисы:

```bash
# Запуск сервера (в одном терминале)
cd diploma-project/server
npm run dev

# Запуск клиента (в другом терминале)
cd diploma-project/client
npm run dev
```

### Просмотр логов

```bash
# Все сервисы
docker-compose logs -f

# Конкретный сервис
docker logs ecosteps_server -f
docker logs ecosteps_postgres -f
docker logs ecosteps_redis -f
```

### Проверка работы Redis

```bash
# Проверить онлайн пользователей
curl http://localhost:3001/api/online-users

# Проверить статистику
curl http://localhost:3001/api/stats

# Подключиться к Redis CLI
docker exec -it ecosteps_redis redis-cli
```

### Тестирование

```bash
# Запустить тесты Redis сессий
docker exec ecosteps_server node test-redis-sessions.js
```

Проект готов к деплою с использованием Docker Compose в продакшн режиме:

```bash

# Остановка
docker-compose -f docker-compose.prod.yml down
```

## ❓ Решение проблем

### Docker не запускается
- Убедитесь, что Docker Desktop запущен
- Проверьте, что виртуализация включена в BIOS

### Порты заняты
- Убедитесь, что порты 5173, 3001, 5432, 6379 свободны
- Остановите другие приложения, использующие эти порты

### Ошибки при сборке
- Очистите Docker кэш: `docker system prune -a`
- Попробуйте заново: `docker-compose up --build -d`
